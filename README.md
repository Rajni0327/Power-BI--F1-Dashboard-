## F1 DashBoard 

An interactive Formula 1 statistics dashboard that provides comprehensive racing data and performance metrics for drivers and constructor teams of 2022.

![F1 Dashboard Preview](https://github.com/Rajni0327/Power-BI--F1-Dashboard-/blob/main/Screenshot%202026-01-28%20214148.png)

## Features

### Driver Statistics
- **Total Wins**: Track the number of race victories for each driver
- **Pole Positions**: Monitor qualifying performance and front-row starts
- **Podium Finishes**: View top-3 finish statistics for all drivers
- **Fastest Laps**: Track record-breaking lap times during races
- **Average Position**: Calculate finishing position averages across the season
- **DNF Tracking**: Monitor "Did Not Finish" statistics

### Constructor Teams
The dashboard includes all official F1 constructor teams:
- **Williams Mercedes**
- **Red Bull Racing RBPT**
- **Mercedes**
- **McLaren Mercedes**
- **Aston Martin Aramco Mercedes**
- **Haas Ferrari**
- **AlphaTauri RBPT**
- **Williams Mercedes**

### Performance Metrics
- **Total Points**: Championship points accumulation
- **Comparative Analysis**: Side-by-side driver comparisons
- **Visual Data Representation**: Bar charts and graphs for easy interpretation
- **Team Selection**: Filter and view team-specific statistics

- **Fan Engagement**: Track favorite drivers and teams throughout the season

## Getting Started

1. Select a constructor team from the team selector
2. View comprehensive driver statistics for that team
3. Compare performance across different metrics
4. Analyze podium finishes, fastest laps, and consistency
